# Iadvize Chat Test

Chat implementation in React/Redux

Use of Webpack as Module Bundler

## Installation:

```
npm install
```

## Building App:

```
npm run build
```

## Start dev server (localhost:3000)

```
npm run dev
```

## Running test

Use of Mocha

```
npm run test
```
